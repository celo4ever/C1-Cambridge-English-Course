# C1 Advanced English Practice App

An interactive web application for practicing C1 Advanced English vocabulary and grammar.

## Content Sources

This application contains exercises and theory based on:
- **Ready for Advanced** (4th Edition) - Macmillan Education
- **English Vocabulary in Use Upper-Intermediate** - Cambridge University Press

## Features

- Interactive exercises with instant feedback
- Theory sections for grammar and vocabulary review
- Progress tracking within sessions
- Mobile-friendly responsive design
- Works offline after initial load

## Topics Covered

### Unit 1: Aiming High
- Modal verbs (might, could, may, can)
- Adverbs of degree and collocations
- Vocabulary: Challenge and Achievement
- Word formation: Noun suffixes

### Vocabulary in Use - Unit 60
- Success, Failure and Difficulty
- Verb + noun collocations (achieve, accomplish, fulfil, etc.)
- Expressions for difficulty (cope with, have trouble, etc.)
- Error correction exercises

## Usage

Open `index.html` in any modern web browser. No installation or server required.
